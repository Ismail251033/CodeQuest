# CodeQuest 🚀

An interactive and gamified coding learning platform built with pure HTML, CSS, and JavaScript.
Learn programming through lessons, quizzes, coding challenges, debugging exercises, XP progression, streaks, achievements, and adaptive learning.

---

## ✨ Features

* 📚 Structured programming courses
* 🧠 Interactive lessons with explanations and quizzes
* 💻 Coding challenges & debugging exercises
* 🎮 Gamified XP and leveling system
* 🔥 Daily streak tracking
* 🏆 Achievements system
* 📈 Progress tracking with localStorage
* 🎯 Adaptive learning & mastery system
* 🌙 Modern glassmorphism UI
* ⚡ Fast and lightweight (no framework required)
* 📱 Responsive design

# 🎮 Gamification System

CodeQuest uses a progression system inspired by games.

## XP System

Users earn XP by:

* Completing lessons
* Solving coding challenges
* Passing quizzes
* Maintaining streaks

## Levels

Every 500 XP increases the user level.

Example:

* Level 1 → Novice Coder
* Level 5 → Code Apprentice
* Level 10 → Junior Developer
* Level 20 → Senior Developer

# 💾 Local Storage

All user data is stored locally in the browser using localStorage.

Stored data includes:

* User profile
* XP
* Progress
* Achievements
* Streaks
* Mistakes
* Settings

No backend required.
